# Financial Sales Performance Analysis

## Project Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, Excel, and Power BI. The objective is to analyze financial sales data, identify business trends, and present insights through an interactive dashboard.

---

## Objectives

- Clean and preprocess the dataset
- Perform exploratory data analysis using Python
- Analyze business metrics using SQL
- Build Pivot Tables and Charts in Excel
- Design an interactive Power BI dashboard
- Generate a professional business report

---

## Dataset

The dataset contains financial sales transactions including:

- Product
- Country
- Segment
- Sales
- Profit
- Units Sold
- Discount Band
- Manufacturing Price
- Sale Price
- Date

---

## Technologies Used

- Python
- Pandas

- MySQL
- Microsoft Excel
- Power BI

## Project Workflow

1. Data Cleaning

## Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Profit Margin KPI
- Units Sold KPI

- Profit by Product

- Scatter Plot
- Interactive Slicers
